# Chat

MijnBureau includes a Matrix-based chat that offers secure, decentralized communication. It is production-ready and suitable for self-hosted deployments. Key features:

- End-to-end encryption for private conversations
- Native clients for desktop, iOS, and Android
- Federation support for decentralized messaging
- Optional message archiving via an archiving bot
